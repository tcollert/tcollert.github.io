Virtual Pet

So, you have some experience under your belt in the care and feeding of a virtual pet. It's time to share that with the community! Time to volunteer! Build upon your Virtual Pet project that allows you to care for multiple pets in a virtual pet shelter! Details In our Virtual Pet project, we created a pet from a VirtualPet class and set up a game loop for user interaction in VirtualPetApp class. For this project, we will be creating a VirtualPetShelter class that allows the user to house and care for a collection of pets. The ideal way to organize your code would be to have your VirtualPetApp call methods from VirtualPetShelter which interact with our VirtualPet objects. We encourage you to use TDD on this project, but it is not required to attain a passing grade. If you feel trying to incorporate testing would be a barrier to your completing the project before the deadline, forgo it.
